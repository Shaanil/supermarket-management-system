# QuickMart — Web-Based Supermarket Ordering System

QuickMart is a web application designed to digitize and streamline supermarket operations. It provides a smooth online shopping experience for customers while offering specialized management dashboards for admins, inventory staff, marketers, and delivery coordinators.

## Features

### For Customers
* **User Authentication:** Secure account creation, login, and profile management.
* **Smart Shopping:** Browse and filter products by name, category, or price with real-time stock availability.
* **Shopping Cart:** Add items, update quantities, and view automatically calculated totals, taxes, and discounts.
* **Order Tracking:** Follow the status of orders from confirmation through to delivery.

### For Management & Staff
* **Admin Dashboard:** Control user permissions, update user roles, and export analytical sales reports in CSV format.
* **Inventory Management:** Add or edit products, monitor stock counts, and get automated low-stock alerts.
* **Payment Management:** Track all transaction logs (Card, PayPal, COD) and handle confirmations or refunds securely.
* **Delivery Coordination:** Assign drivers to orders, manage route tracking, and update dispatch milestones.
* **Marketing Portal:** Create and schedule promotional banners, discount offers, and seasonal campaigns.

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Java (Spring Boot, Spring Security)
* **Database:** MySQL
